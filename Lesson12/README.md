引号的区别
=========

## 知识点

* 字符串中引号的使用
  + 单引号
  + 双引号
  + 引号中的大括号(运算式)

## 实战演习

~~~ruby
# 风言风语NBA
p1 = "我是\n哈登"
p2 = '我是\n卡戴珊'

puts p1
puts p2

# MVP标准：180俱乐部(投篮50%,三分40%,罚球90%)
mvp = "#{50+40+90}"
puts mvp
mvp = '#{50+40+90}'
puts mvp
~~~
